# Identifier Table
Каждый элемент таблицы содержит идентификатор переменной и указатель на дескриптор. Идентификатор переменной - строка длиной до 8 символов. Элементы таблицы упорядочены в алфавитном порядке.
- Дескриптор простой переменной содержит признак определенности переменной и значение переменной.
- Дескриптор вектора содержит максимальный размер и адрес памяти, выделенной под вектор.
- Дескриптор матрицы содержит максимальный размер по каждому измерению и адрес памяти, выделенной под матрицу.
Каждый элемент вектора или матрицы (массива) содержит признак определенности переменной и значение; память под массив выделяется динамически.
Возможные операции с таблицей:
1) Добавить новый элемент
2) Найти элемент по идентификатору
3) Удалить элемент
4) Вывести в консоль содержимое таблицы
5) Вывести суммарный размер переменных в зависимости от их типа, используя класс-итератор. Данная операция осуществляется в однопоточном или многопоточном режиме (по выбору).
Тестирование таблицы осуществляется при помощи фреймворка gtest, сборка проекта - при помощи CMake.
